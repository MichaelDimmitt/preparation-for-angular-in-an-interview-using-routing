## Usage information
```
git clone https://github.com/MichaelDimmitt/preparation-for-angular-in-an-interview-using-routing.git &&
cd preparation-for-angular-in-an-interview-using-routing &&

ng build &&
ng e2e &&
ng test &&
open http://localhost:4200 &&
ng serve
```
